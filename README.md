# Baldi's Basics Plus Name Un-Limiter

## 📝 Functionality

Basically, all this mod does is remove the character limit for name entry, as well as makes you be able to input any characters as long as it: can be rendered by the font, and isn't one of these: < > : " / \ | ? *.
